30 天挑戰 day 1

Posted on 2025-05-11 Edited on 2025-05-18

基本資訊

  • 曲名：空の青さを知る人よ
  • 編號：0571
  • 版本：Hanon (cover)
  • 特點：同名電影主題曲

正文

一開始還真的不知道推什麼好，剛好在歌單中看到了這首，仔細一想我對他還蠻有印象的，於是寫下來了。

說起來一開始會注意到應該是 2:55 開始的那段中四句把一個詞重複五次的歌詞，五次重複都給我不同的感覺，從最開始的絕望，到最後的無奈，搭配下一句的拉高聲音，很有意思。另外後來會喜歡上也一直很想學會的是あいみょん的充滿爆發力的「追いかけている、追いかけている、届け」吧。

不過雖然剛聽到的時候就有查到他是電影主題曲，但高一的時候一直沒有去看，一直到高二選訓才想到這部，然而看到一半大家都跑去聊天了所以我對他的印象停在了 50 分鐘。後來再找了個機會半夜把剩下的 50 分鐘看完，然而許多劇情已經不記得了，因此當下沒有太多感覺。剛剛聽完（以及為了寫這篇）決定再重看了一次電影，開始有了一些感想。

這部電影有蠻多看點的，夢想、感情等等，不過我沒有太多感覺。我反而覺得這部電影講的是「甘心」與「不甘」，在前十分鐘就有這樣的一段：作為姊姊的茜，為了妹妹葵而選擇在盆地中留下，也把自己的夢想留下來了。這樣子的「甘心」有一個很正向的解釋，畢竟對於妹妹的愛以及責任是一個不能否定的課題。而慎之介就這樣到了東京去闖蕩，儘管在意茜卻沒有捎來音訊直到了連茜本人都說生死未卜的地步。到後來才解釋是因為他認為他的事業（演劇的吉他）不能說是成功而不敢告訴眾人。他身上處處都透漏著「不甘」的氣息，但迫於現實，缺乏機遇或什麼的，也只能接受。

甘是一個很微妙的字，一方面甘甜的味道通常給人一個很好的印象，一方面甘心這個詞又似乎給人一種太不進取而稍微負面的感覺。在我拿完 rank 31，差一名就有金牌的時候我也這樣問過自己：「我想要再努力一年還是接受現實？」當下呢，我的回答是接受了現實，無論如何都沒有回去的必要。而現在呢，也許可以算是看到了不一樣的天空了吧。

我很喜歡電影的一句話：「井底之蛙不知道海有多大，但知道天空有多藍。」不管看過多少書，走過多少路，我仍然在不熟悉的地方是隻青蛙，但即使如此，因為我們有同一片天空，這也不是什麼壞事。也許就像歌詞所說的：「全然好きになれなかったんだ、それなのにね、今も似た言葉に身体が動くよ、皮肉な思い出なのさ」儘管當下覺得諷刺，苦澀，但有一天會這樣回甘的吧。

另外，我最終還是改採用了 Hanon 的版本，也許是因為他在唱上面那句時，唱出了回甘的感覺吧，我不知道，連結在這裡。
https://www.youtube.com/watch?v=qqzStAotY14

最後值得一提的是這首歌的 hash value 是 0571，來源是歌名的四個漢字各自的 hash，

  • 空: 空空的，所以是 0
  • 青：藍色，所以是 5
  • 知：發音跟 7 很像
  • 人：1，大概是因為人這個字本身很重要

其他想推薦的歌

  • ヨルシカ - 藍二乗，hash value 1296，感覺也是ヨルシカ裡面比較耐聽的歌，而且歌名有三個解讀方式，非常酷，詳情可參考他的 YT 留言。（hash value 也特意挑了個平方數。）
  • Galileo Galilei - バナナフィッシュの浜辺と黒い虹 (covered by 春茶)，hash value 3997，慚愧，香蕉魚據說是某種暗示但我找了一下沒看得很懂，不過除此之外這首歌非常的順，舒服。
